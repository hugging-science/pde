# PDE Leaderboard

A public benchmark, knowledge hub, and collaboration space for AI-based Partial Differential Equation (PDE) solvers.
This repository brings together code, datasets, experiments, blogs, and community knowledge to track progress in this emerging field.

- Share blogs, tutorials, and insights to help others learn and contribute.
